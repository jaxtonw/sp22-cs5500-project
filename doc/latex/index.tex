 \hypertarget{index_autotoc_md1}{}\doxysection{Series and Parallel Linear Algebra Routines and Functions}\label{index_autotoc_md1}
This library provides Linear Algebra functions and routines performed both serially and in parallel utilizing MPI. The goal is to provide a low-\/end static C++ library for high performance Linear Algebra operations, and study the benefits of parallelization.\hypertarget{index_autotoc_md2}{}\doxysection{To use this documentation}\label{index_autotoc_md2}
